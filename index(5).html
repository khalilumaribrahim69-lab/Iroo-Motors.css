Learning Outcome

JavaScript applied on this webpage

The webpage demonstrates practical JavaScript features below.

1. DOM Manipulation

JavaScript changes page content such as price, time, booking messages, and car details.

2. Form Validation

The booking and contact forms check user input before submission.

3. Event Handling

Buttons, search box, select menus, and forms respond to user actions.

4. Arrays and Objects

Car details are stored in JavaScript objects and displayed when needed.

5. Local Storage

The selected theme and latest booking reference are saved in the browser.

6. Responsive Menu

The mobile menu opens and closes using JavaScript.
