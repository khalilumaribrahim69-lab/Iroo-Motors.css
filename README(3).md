# Iroo Motors JavaScript Webpage Assignment

## Project Title
Application of JavaScript on a Predefined Webpage

## Project Description
This project is a fictional luxury SUV dealership webpage called **Iroo Motors**. The webpage was created to show how JavaScript can be applied to a predefined webpage.

## Files Included
- `index.html` - The main webpage structure
- `styles.css` - The styling and responsive layout
- `script.js` - JavaScript functionality
- `assets/` - Local SUV illustrations used on the webpage

## JavaScript Features Used
1. Current date and time display
2. Light and dark mode toggle
3. Mobile navigation menu
4. Animated statistics counters
5. Car search and budget filtering
6. Car price display using select menu
7. Car details modal popup
8. Booking form validation
9. Contact form validation
10. LocalStorage for saving theme and latest booking reference

## How to Open
1. Download and unzip the project folder.
2. Open the folder.
3. Double-click `index.html` to view the webpage in a browser.

## How to Submit
Submit the full folder or the zip file. Make sure `index.html`, `styles.css`, `script.js`, and the `assets` folder are all included together.
