# Iroo Motors Advanced CSS Webpage

This project is a luxury SUV dealership webpage for Iroo Motors.

## Files Included

- `index.html` - Main webpage structure
- `style.css` - Advanced CSS styling
- `screenshots/desktop-view.png` - Desktop responsive view
- `screenshots/mobile-view.png` - Mobile responsive view
- `iroo_motors_presentation.pptx` - Presentation explaining the CSS improvements

## Advanced CSS Features Used

1. Flexbox layout for the navigation menu, car cards, and service cards.
2. Responsive design using media queries.
3. Mobile-friendly navigation layout.
4. Flexible card layout using `flex-wrap`.
5. Modern styling with gradients, hover effects, spacing, and shadows.

## How to Open

Open `index.html` in a web browser.

## How Responsiveness Was Tested

The webpage was tested using browser Inspect mode with two screen sizes:

1. Desktop/Laptop view
2. Mobile phone view

## Student Note

This project upgrades a simple webpage by applying advanced CSS learning outcomes such as Flexbox and responsiveness.
